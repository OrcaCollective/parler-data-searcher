About the data

This tool lets users search the Parler text post data that was scraped and published by Distributed Denial of Secrets. This dataset includes about 1.6 million posts made on Parler around the time of J6.

How to use this website

If searching for a specific user, use the users search to loosly match a part or the full username or name of a user.

Once you've found the user you're looking for, click the username to open a list of posts by that user.

To do search for post content, use the posts search and fill in the content field. This will partially match any of the content fields available in the dataset.
